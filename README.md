# Daily Practice

React + Vite MVP for a Korean vocabulary Daily Practice learning flow.

## Setup

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## Notes

- The app currently uses mock data only.
- Mock data is separated like future database tables under `src/mock`.
- The Unit 1 MVP contains the 20 questions from the latest learning plan.
- Review Builder Agent logic evaluates six skills and creates session-only personalized review questions.
